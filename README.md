some notes when I am taking [Codecademy Learn the Command Line](https://www.codecademy.com/learn/learn-the-command-line) course.

1. Directory:

    - `cd` : Change Direcotry（打开当前文件夹）
    - `mkdir` : MaKe DIRectory (新建文件夹)
    - `touch` : 新建文件
